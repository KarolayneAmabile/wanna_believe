# I Want To Believe 🍃
Uma rede social experimental feita como trabalho final para disciplina de Programação para Dispositivos Móveis.

## 🚀 Funcionalidades
- **Autenticação completa:** Registro, Login e Logout integrados ao Firebase Auth.
- **Feed em Tempo Real:** As publicações aparecem instantaneamente para todos os usuários assim que são criadas, graças aos *Snapshot Listeners* do Firestore.
- **Interações Sociais:**
  - Sistema de "Likes" (Curtidas).
  - Comentários em tempo real em cada post.
  - Exclusão de posts.
- **Perfil do Usuário:**
  - Edição de nome.
  - Upload de foto de perfil via Firebase Storage.
---

## 🛠️ Tecnologias Utilizadas
- **Jetpack Compose:** Para uma interface declarativa e moderna.
- **MVVM (Model-View-ViewModel):** Arquitetura que separa a lógica de negócio da interface.
- **Firebase:**
  - **Firestore:** Banco de dados NoSQL para posts e comentários.
  - **Authentication:** Gerenciamento de usuários.
  - **Storage:** Armazenamento de imagens.
- **Coil:** Carregamento de imagens assíncrono e eficiente.
- **Coroutines & Flow:** Para lidar com operações assíncronas e fluxo de dados em tempo real.
